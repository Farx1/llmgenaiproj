# ESILV Smart Assistant - Frontend

Next.js frontend for the ESILV Smart Assistant chatbot.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set environment variables (optional):
```bash
# Create .env.local
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Chat Interface**: Interactive chat with the ESILV Smart Assistant
- **Model Selection**: Choose between different LLM models (llama3, mistral:7b)
- **Document Upload**: Upload documents to enhance the knowledge base
- **Admin Dashboard**: View system statistics and collected contacts
